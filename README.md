# 相关的Emoji在程序开发中的应用

| Emoji | 含义 | 适用场景 |
|-------|------|--------|
| ✨ `:sparkles:` | 新功能（Feature） | 添加新功能、模块或接口 |
| 🐛 `:bug:` | 修复 Bug | 修复程序中的错误或异常行为 |
| 🚀 `:rocket:` | 部署/上线 | 用于部署、发布版本等 |
| 📦 `:package:` | 打包/依赖 | 更新依赖、构建配置、包管理 |
| 💄 `:lipstick:` | UI/样式改进 | CSS、UI 微调、视觉优化 |
| 🎨 `:art:` | 代码格式/美化 | 代码风格调整、格式化（不改变逻辑） |
| 🔧 `:wrench:` | 配置文件修改 | 修改配置、环境变量、工具设置等 |
| 📝 `:pencil:` | 文档更新 | README、注释、文档等 |
| 🔒 `:lock:` | 安全相关 | 修复安全漏洞、权限问题等 |
| ⚡️ `:zap:` | 性能优化 | 提升性能、减少资源消耗 |
| ♻️ `:recycle:` | 代码重构 | 重构代码结构，不改变外部行为 |
| 🧪 `:test_tube:` 或 🧪 `:alembic:` | 测试相关 | 添加/修改测试用例 |
| 💥 `:boom:` | 破坏性变更（Breaking Change） | 不兼容的 API 变更 |
| 🗑️ `:wastebasket:` | 删除代码或文件 | 移除无用代码、文件等 |
| 🌐 `:globe_with_meridians:` | 国际化/本地化 | 多语言支持、i18n 相关 |
| 🚑 `:ambulance:` | 紧急修复 | 严重 bug 的热修复 |

参考：[Gitmoji 表情符号](https://gitmoji.dev/?spm=5176.28103460.0.0.460b75512aLIef)
